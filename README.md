# Student-SQL

## SQL Student Management Project

A practice project to manage student records using SQL.

Overview

This project demonstrates creating, updating, deleting, and reporting on a student database. It focuses on core SQL operations for learning and portfolio purposes.

Database Structure

Table: students

student_id (INT, PK)

first_name (VARCHAR)

last_name (VARCHAR)

program (VARCHAR)

average (DECIMAL)

enrollment_date (DATE)

Key Operations

Create Table – Define the students table with constraints.

Insert Records – Add new students.

Update Records – Modify student data.

Delete Records – Remove graduated students.

Reports – Count students per program.
